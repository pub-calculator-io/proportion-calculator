The ratio calculator finds a missing ratio number based on the given proportion. The calculator can also scale ratios, enlarging or shrinking them.

[https://www.calculator.io/proportion-calculator/](https://www.calculator.io/proportion-calculator/)

== Usage ==

Add the Proportion Calculator  shortcode to your page, post or sidebar:

`[ci_proportion_calculator]`

Add the following code to your template where you would like the Proportion Calculator  to appear:

`<?php display_ci_proportion_calculator(); ?>`

== Installation ==

1. Upload the Proportion Calculator  /ci_proportion_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Proportion Calculator  plugin through the "Plugins" menu in WordPress.
